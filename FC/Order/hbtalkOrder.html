HB TALK 신청안내

  • HB TALK 학습을 신청할 수 있는 페이지로 회원 별 [신청]버튼을 클릭 하면 신청이 가능합니다.
  • 아래 회원 리스트는 HB TALK 신청이 가능한 회원이며, 신청조건은 아래와 같습니다.
    • 주문 시스템을 이용하여 주문한 최근 6개월 교재에 대한 수업만 신청가능
    • 당월 주문한 교재의 경우, 교재주문 지사/본사 승인 후 신청가능
  • 신청이 완료되면 신청내역에서 현황을 볼 수 있으며, 수정, 취소가 가능합니다.
  • ID 발급을 하지 않은 학생은 HB Talk 신청이 불가능합니다. 회원ID 발급 후 신청해주세요.
  • 매월 1일 ~ 10일 : 전 달 HB TALK 수강신청 회원만 신청가능
  • 매월 11일 ~ 31일 : 모든 회원 HB TALK 수강 신청가능(2015.01부터 시행)
  • HB TALK은 해법영어교실 회원에게만 주어지는 학습 서비스이며, 회원 휴/탈회시에는 해당서비스를 이용하실 수 없습니다.
    회원 휴/탈회로 인해 발생하는 미수업분은 환불 처리되지 않으니, 수업신청에 각별히 유의해주시기 바랍니다.
  • 익월 과금 교재 주문 시 HB Talk 서비스를 이용하실 수 있으므로, 수업신청 전에 교재주문내역을 확인해주시기 바랍니다.
